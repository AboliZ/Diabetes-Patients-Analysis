# Diabetes Patients Analysis

## Overview

Predicting diabetes in female patients of Pima Indian heritage based on diagnostic measurements.

## Dataset Features

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI (Body Mass Index)
- Diabetes Pedigree Function
- Age
- Outcome (0 for no diabetes, 1 for diabetes)
